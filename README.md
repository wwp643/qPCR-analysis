# 🔬 一键式 $\Delta \Delta C_t$ 相对定量分析工具

## 概述

本项目提供了一个基于 HTML 和 JavaScript 的**独立网页工具**，用于简化 $\text{qPCR}$ (定量聚合酶链式反应) 数据的相对定量分析。该工具采用标准的 $\Delta \Delta C_t$ (Delta Delta Ct) 方法，能够自动将长列表原始数据转换为三重复表格，并计算最终的相对表达量 (Fold Change)。

## ✨ 核心特色

* **一键式操作：** 粘贴原始数据后，通过选择对照组和内参基因，即可自动完成所有计算。
* **输入兼容性高：** 支持仪器导出的原始数据格式，可兼容 `Ct` 或 `expr` 作为 Ct 值列名。
* **动态参数选择：** **内参基因**和**对照组前缀**的选项会自动根据您粘贴的数据内容动态生成。
* **详细计算过程：** 结果清晰展示了平均 Ct 值、$\Delta C_t$、$\Delta \Delta C_t$ 和最终的 $2^{-\Delta \Delta C_t}$ 结果。

## 🚀 使用指南

### 1. 访问工具

您可以直接在浏览器中打开 `qpcr_analysis_tool_v2.html` 文件来使用该工具。

### 2. 数据准备与输入

请从您的 $\text{qPCR}$ 仪或数据处理软件中导出包含以下三列的**长列表数据**，并将其粘贴到文本框中（列之间使用制表符 Tab 分隔）：

| 列名 | 说明 | 示例值 |
| :--- | :--- | :--- |
| `SampleName` | 样本名称（例如：Con-1, Treat-A） | `Con-1` |
| `TargetName` | 目标基因或内参基因名称 | `GeneX` |
| `Ct` 或 `expr` | 循环阈值 (Ct Value) | `22.30` |
